My Role

Jupyter is an open source organization that builds an interactive data science IDE used by over 30 million people around the world. I worked there as a hybrid UI/UX Designer and Front-end Engineer for 9 months.

JupyterLab

The Project

My main job at Jupyter was to help build the next generation JupyterLab web application. The existing Jupyter Notebooks that all our users knew and loved allowed users to wirte and run code inline in an extremely visual and interactive way. JupyterLab builds around the notebook by adding commonly used developer tools (e.g. a terminal, file browser, text editor, etc.) in a multiwindow interface. Our team’s job was to research, design, and develop these tools (called widgets) into JupyterLab as well as try to improve the overall interface. The entire app was built with javascript using a custom framework called Phosphor.JS.

Widgets and parts of Jupyterlab I worked on:
- Top navigation bar
- Terminal
- Image Viewer
- Redid File Browser
- Notebook widget toolbar
